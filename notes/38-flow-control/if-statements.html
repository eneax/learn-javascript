Truthy values

When a truthy value runs, it'll equate to true

  • 1
  • -10 (zero is falsy, but all other numbers are truthy)
  • full string (any string with content in it)
  • a string of "0"
  • empty array
  • empty object

Falsy values

When a falsy value runs, it'll equate to false

  • 0
  • undefined
  • a variable set to `null`
  • NaN
  • empty string
